Detailed Description

A GL window with numerous built-in, thread-safe drawing operations.

Canvas provides an easy-to-set-up, easy-to-use class for drawing various shapes.

With libpng and libjpeg, Canvas also supports the drawing of images.

On top of being easy to use, Canvas is also thread-safe, so any number of images may be drawn at once.

void Canvas::end ( )

Stops the Canvas.

This function stops rendering the Canvas. This does not close the Canvas; it will stay open. To rejoin the threads and complete the close, close() must be called. end() is not required to close the Canvas, closing the window by hitting escape or with the window manager does the same thing.

Once end() has been called, either manually, by the window manager, or by pressing escape, getIsOpen() will return false, and your computation should exit.
