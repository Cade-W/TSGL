Blends two colors with a given bias towards the latter.

This function blends two ColorFloat structs together by taking a linear interpolation between the two and returns the result as a new ColorFloat.

Parameters
c1A ColorFloat.
c2Another ColorFloat.
biasA bias between 0 and 1 inclusive. A bias of 0 returns c1, a bias of 1 returns c2, and a bias in between returns a linear interpolation.
Returns
A ColorFloat linearly interpolated between c1 and c2 using the given bias as a weight.

Returns an HSVA color with a hue dependent on the number of sections.

This function returns a ColorFloat whose hue is calculated from the provided section number and the total number of sections. This function is used for creating a chromatic gradient from one part of the spectrum to another.

Parameters
sectionInteger specifying the current section.
sectionsInteger specifying the total number of sections.
valueValue component, between 0 and 1 inclusive.
alphaAlpha component, between 0 and 1 inclusive.
Returns
A ColorFloat with a hue calculated as 6.0f/sections*section, saturation of 1.0, and the given value and alpha components.
