\subsection{Detailed Description}
Draw an arbitrary polygon with colored vertices. 

\hyperlink{classtsgl_1_1_colored_polygon}{Colored\+Polygon} is a class for holding vertex data for a triangle strip with colored vertices.

Vertices are drawn in triangle strip format, where the first three vertices make up the first triangle, the next vertex plus the previous two make up the second triangle, and so on.

This method is optimized for long lists and offers a marked improvement over drawing individual \hyperlink{classtsgl_1_1_triangle}{Triangle} instances. \begin{DoxyNote}{Note}
The \hyperlink{classtsgl_1_1_colored_polygon_a70c4e2bd5a96623f1d8a1d0206ececed}{add\+Vertex()} method must be called the same number of times as specified in the constructor. 

Calling \hyperlink{classtsgl_1_1_colored_polygon_a70c4e2bd5a96623f1d8a1d0206ececed}{add\+Vertex()} after all vertices have been added will do nothing. 

Calling \hyperlink{classtsgl_1_1_colored_polygon_a6b93c1fa689dbee7592faeda0cf13933}{draw()} before all vertices have been added will do nothing. 
\end{DoxyNote}
