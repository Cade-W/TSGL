Detailed Description

Draw multiple lines chained together.

Polyline is a class for holding vertex data for multiple lines whose endpoints are connected.

This method is optimized for long lists and offers a marked improvement over drawing individual Line instances.

Note
The addVertex() method must be called the same number of times as specified in the constructor.
Calling addVertex() after all vertices have been added will do nothing.
Calling draw() before all vertices have been added will do nothing.
